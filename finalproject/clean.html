Using Organizational Tags

When you are programming it helps to know where sections of code are. It is also nice to know which section of code does what, especially when doing updates. Using tags such as "Div", "Section", "Nav", etc can be a valuable resource in doing exactly this. The div tag or "division" can be used to identify sections of content. This is the same for Section and Nav. Though they are "spelled" differently they all perform relatively the same function. Although when reading them they can help identify what that specific section of code is used for. Take nav for instance. This tag indicates that that section of code is for navigation across the website. While Picture indicates that it has to deal with, well pictures.

Comments

comments can help you remember why certain things have been done. An example of this can be if you use single quotes in your code and you normally use double quotes. A comment can be used to remind yourself in the future why you did so.

Using CSS

CSS can be a lot of fun to help make everything on a webpage be more colorful and exciting. Although, CSS can also be used to help organize content on your webpage. For instance when developing a header or footer CSS can come in very handy to keep content where you want it and how you want it.

Notes and Classes

The only thing about CSS is that on default it applies the attributes to every tag. This is where classes come in. When a class is applied to a tag that class then can be assigned attributes. Though these attributes will only be applied to the class and not the tag.

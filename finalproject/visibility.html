Fonts

Fonts are a fun topic. They can be used to help content be easier to read or they can cause text to be entirely unreadable all together. When used right though, they can add texture and character to your website. Now this is all fun and games but as previously mentioned if you use a wierd or creative font it can cause difficulty to the user of your website.

Color Matching

Colors can be tricky, you need colors that work well with each other and are not to similar though. Think of a typucal book, using white pages and black text. The colors contrast each other in such a way so that a reader can read the words. This same priinciples applies to websites. Another example that you can think of is a white and a grey color on top of each other. In this situation the colors may be difficult to see because of the similarities.

Headings

Believe it or not there is an order at which you use headings. For example, just because the h1 header is the bigest doesn't mean it can be just thrown anywhere. Headers help create order on a webpage. Starting with h1, headers are able to work down to h6. These headings help indicate sections of content. For instance h1 is typicaly used as the title or head of the page. Then the headers are used as necesary. One such situation where headers are recommended is at the beguining of each section tag.
